# Random Quote Generator

A modern and responsive web application that fetches and displays random quotes dynamically using a public API and JavaScript Fetch API.

---

## 🚀 Live Features

- Fetch random quotes dynamically
- Responsive design for mobile and desktop
- Loading state implementation
- Error handling for failed API requests
- Interactive and clean user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

---
